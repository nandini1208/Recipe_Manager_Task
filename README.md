# Recipe Manager Web App

A full-stack recipe management web application built with HTML, CSS, and JavaScript using browser localStorage.

## Features

- Create, read, update, and delete recipes
- Search recipes by title and filter by difficulty
- Responsive design for desktop and mobile
- Form validation with error messages
- Data persists in browser localStorage

## How to Run

1. Download all project files
2. Open `index.html` in a web browser
3. The app will load with sample recipes automatically
